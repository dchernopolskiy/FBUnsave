FB Marketplace Saved Filter - Chrome extension to filter and search your Facebook Marketplace saved listings.

Features:

Hide sold and pending items from your saved listings
Search through all saved items with auto-scroll to load everything
Navigate between search results with prev/next buttons or keyboard shortcuts (, / .)
